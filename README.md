# axi-iccare

This repository contains a Python project designed to control an AxiDraw using voice commands via speech recognition.

## Features
- Real-time speech recognition using a microphone.
- Commands to draw shapes like circles and squares using AxiDraw.
- Easy to expand with additional drawing commands.

## Prerequisites
- AxiDraw hardware and software.
- Python 3.x installed on your machine.
- Install the required Python libraries:
  ```
  pip install SpeechRecognition pyaxidraw
  ```
- Ensure that your microphone is functioning properly for speech input.

## How to Use
1. Connect your AxiDraw machine.
2. Run the Python script `voice_control_axidraw.py`.
3. Speak commands like:
   - "Draw a circle."
   - "Draw a square."
4. To exit, say "stop".
