## How to Use

1. Connect your AxiDraw machine.
2. Run the Python script `voice_control_axidraw.py` with the desired language option. For example:
   ```plaintext
   python voice_control_axidraw.py --language fr-FR
   ```
   Replace `fr-FR` with your desired language code (default is English "en-US").
3. Speak commands in the selected language, such as:
   - "Draw a circle."
   - "Draw a square."
4. To exit, say "stop."
